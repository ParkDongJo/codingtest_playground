# 강의 해설을 보기 전 돌려보기

- n을 받아서 자리수를 구한다. (m 이라고 지칭)
- m의 자리수 만큼 특정 연산을 수행한다.
- 연산

```c
while (flag) {
  // 모듈러 계산을 할 대상을 만든다.
  for (i = m; i > 0; i--) {
    target += 10 ^ i
  }
  // 모듈러 계산을 하고
  if (target % n == 0) {
    // 통과를 하면, 상위 루프를 멈춘다
    flag = false;
  } else {
    // 통과하지 못하면, 더 큰 계산 대상을 만들게끔 m을 올린다.
    m += 1
  }
}

```

# 해설

... 아오... 진짜 머리가

정수론의 모듈러 연산 법칙

> (a+b) % n = (a%n + b%n) % n
> (a*b) % n = (a%n * b%n) % n

끝에 가서 모듈연산한 것과, 그 전에 모듈연산을 꾸준히 한 것과 동일하다.

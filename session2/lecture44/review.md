# 강의 보기전 먼저 생각해보기

[조건]
테스트 케이스 100
의상의 수 30개 미만

[중요한 전제]
의상 종류가 같으면 동시에 사용 불가

의상 종류로 군집을 만들어 놓고! (의상 종류로 그룹핑하는 로직을 생각해두기)
각 군집에서 하나씩만 빼와서 조합하여 만들 수 있는 경우의 수를 구하기

수식을 만들어보면
의상의 전체 갯수 + 군집으로 만들 수 있는 조합 갯수 (A 군집 갯수 \* B 군집갯수)

# 강의를 보면

의상의 수 군집을 하고, 각 군집의 조합 경우의 수를 구하기 위해 곱한 후에,
-1 을 해주는데 여기서 1가지를 빼는 이유는, 아무것도 안 입은 경우를 뺀다.

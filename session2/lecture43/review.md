# 학습 하기 전 나

배열의 크기나, 성능에 제한이 있어보이지 않는다.

```
Z C C D X
K Q M N B
U O W Z Y
F C O I J
P A Q Z T
```

dict.txt 에 있는 단어목록들

1. dixt.txt 목록을 배열에 담고 있어야 한다.
2. 단어 목록 배열을 돌면서 단어를 빼오면, 매트릭스의 1,1 부터 돌아가면서 첫번째 글자를 찾아야한다.
3. 첫번째 글자를 찾으면 가장 상위 루프는 멈추고

- 그 기준 단어를 중심으로 전 방위로 index 로 접근하면서, n번째 글자가 있는지를 루프를 돌면서 찾아야 한다.
- 이때 한번 비교한 글자는 index 를 제외 시켜야한다.
- 만약 한 글자라도 성립이 안되면, 더이상 연산을 멈추고 가장 상위 루프가 다시 돌면서 체크 해야한다.

이때 범위를 체크할때는 5*5 매트릭스로 보는 것이 아니라, 7*7 매트릭스로 봐야한다

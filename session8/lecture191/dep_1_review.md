# 문제 해설에서 DP 를 설명하는 것

- 그 문제의 주요 상태들을 뽑고
- 그 상태들간의 관계를 통해 특정 식을 도출한 다음
- 메모이제이션을 더한다

```c++
int go(int idx, int m, int pos) {
  // 특정 결과를 계속적으로 메모이제이셔 하는
  if (dp) return dp

  // 이렇게 주요 상태들의 관계 식을 한 후
  // 아래는 단순히 완탐
  return max(
    go(idx + 1, m, pos),
    go(idx + 1, m - 1, pos == 1 ? 2 : 1)
  )
}
```

# Session 8 - 동적 프로그래밍 (DP)

## 개요
동적 프로그래밍의 탑다운/바텀업 접근법과 실전 응용 문제를 다루는 세션입니다.

## 강의별 정리

### 강의 191 - DP 기초와 응용
| 파일 | 알고리즘 | 설명 |
|------|----------|------|
| fibo_topdown.cpp / .js | 탑다운 DP (메모이제이션) | 재귀 + dp 배열로 중복 계산 방지, `dp[n] != -1`이면 캐시 반환 |
| fibo_bottomup.cpp / .js | 바텀업 DP (반복문) | for 루프로 아래에서 위로 순차 계산, 재귀 오버헤드 없음 |
| dp_1.cpp / .js | 3D DP (사과 잡기) | `dp[시간][나무][이동횟수]` 3차원 상태 공간, XOR로 나무 번호 전환 |
| num_play.cpp / .js | 조합론 DP (숫자 분할) | 정수 n을 1,2,3의 합으로 나타내는 경우의 수, 코인 체인지 변형 |

## 핵심 개념

### 탑다운 vs 바텀업
| 구분 | 탑다운 (Top-Down) | 바텀업 (Bottom-Up) |
|------|-------------------|-------------------|
| 방식 | 재귀 + 메모이제이션 | 반복문 + DP 테이블 |
| 장점 | 필요한 부분만 계산, 직관적 | 재귀 오버헤드 없음, 빠름 |
| 단점 | 재귀 스택 오버헤드 | 모든 값 계산 필요 |

### DP 문제 풀이 패턴
1. **상태 정의**: dp 배열의 인덱스가 의미하는 것 정의
2. **점화식 도출**: 현재 상태를 이전 상태들로 표현
3. **기저사례 설정**: 초기값 설정 (dp[0], dp[1] 등)
4. **계산 순서 결정**: 탑다운 or 바텀업 선택

# 고지식하면, 고생한다

해당 문제는 counting star 기법으로 풀려고 했다.
그래서 중앙을 잡고, 반만 반복문을 돌아서 카운팅을 하면서 체크를 하려고 했다.

하지만.. 실제로 문제 풀이는 매우 간단하다.

뒤집어도 글자가 같다는 특징을 활용해서, 그냥 입력값과 비교하면 끝이다.

# 배운점

- 센스가 필요하다. 센스를 가지고 태어나지 않았다면.. 곡씹어서 감각을 만들 수 밖에 없을 것 같다
- 뒤집는다 라는 기법을 기억해야할 것 같다. 센스를 가지기 전 까지는 '뒤집으면 어떨까?' 라는 질문을 계속 하면서, 그게 유용한 문제 유형들을 기록하는 수 밖에 없을 것 같다.

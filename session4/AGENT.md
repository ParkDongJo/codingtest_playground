# Session 4 - 완전탐색과 백트래킹

## 개요
완전탐색(Brute Force), 백트래킹, 조합을 활용한 문제 해결, BFS/DFS 응용 문제를 다루는 세션입니다.

## 핵심 개념
- **완전탐색**: 모든 경우의 수 나열 (O(2^n), O(n!))
- **백트래킹**: 가지치기로 탐색 공간 축소
- **조합**: nCr 재귀 생성, 실전 문제 응용
- **BFS 응용**: 다중 시작점 최단 거리
- **DFS 시뮬레이션**: 연결 성분 기반 상태 변화 반복

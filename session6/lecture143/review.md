# 강의 듣기 전 생각

1. 무식하게 풀수 없다. 조건이 1,000,000,000 이상의 범위이다

2. DP 는 일단 모르겠다

3. 그리디?.. 명제를 생각해봤다

- `이전 끝선의 위치에 다음 시작선의 위치를 빼서, 0 보다 작게 나오면 겹친다`고 판단한다.
- 두 선을 더하고 겹친 길이를 뺀다. 그리고 그것을 저장해둔다.
- 이 작업을 계속 해 간다.

# 강의 해설

라인 스위핑 기법으로 풀어간다.

- 라인 스위핑은 입력 값을 모두 받고! 우선 정렬을 해야한다.
- 데이터는 각 선들의 from, to 를 저장해둔다.
- 정렬한 상태에서 첫 선의 from, to를 start, end 로 잡는다.
- 겹쳐진 상황에서는 더 뒤의 to 값이 end 로 갱신된다.
- 겹쳐지지 않았다면, 새로운 선의 from, to 를 start, end 로 재셋팅 한다.

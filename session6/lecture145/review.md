# 강의의 해설
로직을 생각해보고 -> 반례 생각하기 -> 반례가 나오면 -> 방향을 빠르게 틀기

- d(날짜를) 작은수 부터 오름차순으로 정렬한다. p 돈은 뒤죽박죽이다
- 최대 == 최소 가장작게 또는 최대를 가장 크게
- 하나씩 PQ(오름차순 우선순위 큐)에 넣는다. 2개 갯수가 되면 최소는 빼버린다.


d 기준으로 오름차순 또는 내림차순
PQ 를 오름차순 또는 내림차순

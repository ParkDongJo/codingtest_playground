# Session 1 - 재귀와 경우의 수

## 개요
재귀함수의 기본 개념과 경우의 수(순열, 조합)를 다루는 세션입니다.

## 강의별 정리

### 강의 9 - 재귀함수 기초
| 파일 | 알고리즘 | 설명 |
|------|----------|------|
| fact.cpp / fact.js | 팩토리얼, 피보나치 | 재귀 vs 반복문을 이용한 팩토리얼 계산, 재귀를 이용한 피보나치 수열 |

- 팩토리얼: `n! = n * (n-1)!`, 기저사례 `n=0 -> 1`
- 피보나치: `fibo(n) = fibo(n-1) + fibo(n-2)`, 기저사례 `n<=1 -> n`

### 강의 10 - 순열 (Permutation)
| 파일 | 알고리즘 | 설명 |
|------|----------|------|
| perm.cpp / perm.js | next_permutation | 오름차순 정렬된 배열에서 다음 순열을 반복 생성하여 nPr 구현 |

- `nPr = n! / (n-r)!` : 순서가 있는 경우의 수
- 배열 정렬 후 `do-while`로 모든 순열 열거

### 강의 12 - 조합 (Combination)
| 파일 | 알고리즘 | 설명 |
|------|----------|------|
| combi.cpp / combi.js | 재귀 조합, 삼중 반복문 | start 인덱스 기반 재귀 조합 생성, 반복문을 이용한 조합 |

- `nCr = n! / ((n-r)! * r!)` : 순서가 없는 경우의 수
- 재귀: start 인덱스로 중복 방지, `v.length == k`이면 출력
- 반복문: 삼중 for 루프로 3개 선택 조합 생성

## 핵심 개념
- 재귀함수의 기저사례(base case) 설정
- 순열과 조합의 차이 (순서 유무)
- 재귀 vs 반복문 변환

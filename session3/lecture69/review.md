# 해설 강의 듣기 전 직접 방향 정해보기

N이 100이하라고 했고, 사각형은 최대 100열, 100행이라고 했으니, 완탐으로 접근을 해볼만하다.

- DFS 로 하나씩 지점을 다 탐색을 해야한다.
- N 보다 큰 지점만 DFS 로 체크를 한다.
- 체크가 이미 된 지역은 스킵한다.
- 체크를 더 이상 할수 없을 시 cnt 을 올린다.
- 모든 체크가 그 회차에서 끝났을 시, max 를 갱신한다.
- 최종 max 를 반환한다.

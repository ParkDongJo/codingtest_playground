# 강의 듣기전에 직접 생각해보기

해당 로직을 어떻게??

- 전체에서 점차적으로 4등분 해야한다.
- 왼위 -> 오위 -> 왼아 -> 오아 로 돌아가면서 4등분 할수 있는지를 판단해야한다.
- 재귀로
  - 더이상 4등분 할 수 없을 때는 왼위 -> 오위 -> 왼아 -> 오아 가면서 값을 꺼내오고 반환한다
  - 4등분 할 수 있을 때는 재귀를 돈다

4등분 한다는 것은?

- N \* N 일때 이를 각각 2등분 해서 재귀로 넘겨야 한다.
- N / 2 가

  - 1 이면 더이상 4등분 할 수 없다
  - 1 이상이면 더 4등분 할 수 있다

- 진입하면 먼저, 전체가 다 동일한 값인지 확인해야한다.

  - DFS 로 정해진 범위(N / 2) 안에서, 체크를 한다.
  - 탐색 시작점과 끝점을 알아야 한다.
  - 동일한 값이 아니면, 멈추고 4등분 한다.
    - 기존 탐색을 초기화 한다.
  - 저니간 4등분 할 수 없다면, 값을 만들어서 반환한다.

하.. 스바 어렵다.. 복잡하다

# 강의 해설

- 행위(로직은 반복된다)
  - 0,1 판단 -> Z 로 탐색 -> 결과물 만들기
- 재귀

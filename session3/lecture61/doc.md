2. 탐색 방법 선정

- 큰돌님은 좌표를 (y, x) 로 하더라.. 이유는 취향차이라고 했음
- 내 생각에는 인접행렬 기준 (세로, 가로) 의 관점이 편한거 같아서 하는듯

- 의도하는 방향대로 탐색을 한다면

```c++
const int dy[] = {-1, 0, 1, 0};
const int dx[] = {0, 1, 0, -1};

for(int i = 0; i < 4; i++){
  ny = y + dy[i];
  nx = x + dx[i];
}
```

즉, 위에서 dy, dx 는 우리가 키보드의 방향키 같은 역할이라고 생각하면 된다.

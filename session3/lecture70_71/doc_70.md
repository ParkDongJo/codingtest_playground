# 반례를 찾는것

제출 했을 시, 숨어있는 TC 를 생각해야한다.

# 반례 생각하는 팁

주어진 테스트 케이스는 주로 그 사이 범위에 있는 것들로 이뤄진다.

우리는 숨겨진 TC 를 생각할 때 끝 영역에 있는 영역들을 생각해야한다.

- 최소 / 최대
- 없거나 / 있거나

# 사례

69강 안정영역

- 노트에 '아무 지역도 물에 잠기지 않을 수 있다'
- 문제의 범위가 '높이 1이상 100 이하' 로 되어 있다.

만약 문제의 범위만 생각하고 범위를 1이하 100이상으로 해버리면, 아무것도 잠기지 않은 상태는 체크하지 못하게 된다!

그렇다면 0이하 100이상으로 해서 0(비가 아예 오지 않았을 시)를 체크하던가 해야한다.
